## Scanner와 system.in

### 실습 ScannerLab1

1. 클래스명 : ScannerLab1

2. 표준입력으로 숫자 두 개와 연산자 문자 한 개를 입력받아서 각각 변수에 저장한다.

   첫 번째 숫자를 입력하세요 : 

   두 번째 숫자를 입력하세요 : 

   연산자(+,-,*,/)를 입력하세요 : 

   nextInt( ), nextInt( ), next( )

   nextLine( ) -> 숫자 입력시 다시 숫자로 변환하는 작업 필요

   ​					int Interger.parseInt(String)

3. switch문 사용

   입력된 연산자가 "+"이면 입력된 두 개 숫자의 덧셈을 처리한다.

   입력된 연산자가 "-"이면 입력된 두 개 숫자의 뺄셈을 처리한다.

   입력된 연산자가 "/"이면 입력된 두 개 숫자의 나눗셈을 처리한다.

   입력된 연산자가 "*"이면 입력된 두 개 숫자의 곱셈을 처리한다.

   +,-,*,/ 중에 하나가 입력되지 않으면 "+,-,*,/를 입력해라" 이라는 메시지를 출력한다.

4. 결과는 다음과 같이 출력한다.

   xx와 yy의 X연산 결과는 zz입니다.

```java
package day8;
import java.util.Scanner;

public class ScannerLab1 {
    public static void main(String[ ] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("첫 번째 숫자를 입력하세요 : ");
        int wakeUp = sc.nextInt();
        sc.nextLine();
        
        System.out.print("두 번째 숫자를 입력하세요 : ");
        int getUp = sc.nextInt();
        sc.nextLine();
        
        System.out.print("연산자 (+,=,*,/)를 입력하세요 : ");
        String callMe = sc.next();
        
        int good=0; //초기화를 안해줘서 결과가 자꾸 에러가 남 =>이로 인해 다른부분 다 완벽하게 했으나 늦게 제출하게 됨
        switch(callMe) {
            case "+" :
                good=wakeUp+getUp;
                break;
            case "-" : 
                good=wakeUp-getUp;
                break;
            case "*" : 
                good=wakeUp*getUp;
                break;
            case "/" : 
                good=wakeUp/getUp;
                break;
            default :
                System.out.println("+,-,*,/를 입력하세유");
                sc.close();
                return; //메인 메서드 안에서 리턴문 사용시 프로그램을 끝내겠다는 의미
        }
        System.out.println(wakeUp+"와 "+getUp+"의 "+callMe+"연산 결과는 "+good+"입니다.");
        sc.close();
    }
}
```

- 자꾸 초보적인 실수로 인해 나머지 모든 부분을 정답과 근접하게 했더라도 제출 시간이 느려지게 됨.
